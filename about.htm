About

About Like Up:
This project, built at HackMIT 2018, aims to provide an inline like-prediction and social-network optimization to users as the post on common social media platforms like Instagram, Facebook, and Twitter.
How it works:
Using publicly available social media data, we created a set of which contained the following: picture, caption, likes, friends/followers. Then, we trained a neural net to predict number of likes.
Next Steps:
We would like to train this on more data, and include different categories of pictures.
